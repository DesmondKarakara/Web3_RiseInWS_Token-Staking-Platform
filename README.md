# Token Staking Platform

## Project Description

Token Staking Platform is a basic Soroban smart contract built on the Stellar blockchain. It allows users to stake tokens, track their staked balance, and withdraw tokens securely. This project is a simple foundation for a decentralized staking application.

## What it does

This smart contract lets users:
- Stake tokens into the platform
- Withdraw staked tokens
- Check their individual staking balance
- View the total amount staked across the platform

## Features

- Built with Soroban SDK for Stellar
- User-authenticated staking and withdrawal
- Per-user staking balance tracking
- Total staking pool tracking
- Simple and lightweight smart contract design
